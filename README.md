## Dependencies

- g++
- cmake

## Compiling with CMake

```shell
$ cd [rb_tree] #_go to the project folder_
$ mkdir build #_create a build folder_#
$ cd build
$ cmake .. #_create makefiles. Remember, to use cmake you should have the CMakeLists.txt file_ )
$ make #_run make_
```

## Running

```shell
$ ./test #_run executable_
```
